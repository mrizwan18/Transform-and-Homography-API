Transform API
Examples

FIRST

Generate a 3D face mesh using VRN or PRNet or any other modeling software you prefer.

THEN

Write that .obj file contents in an array. Make a JSON object with 3 params i.e. 1)image array you formed earlier, 2)target image array, 3)array of 3 params [distance from camera int, degree of eye left/right int, degree of head up/down int]
{
"source":"[v 1 1 5 0.4 0.3 0.2 ...... f 0 2 1]",
"target": "[0.5 0.1 0.04 ...... 0.8 0.11 0.01]",
"params":"[300 50 0]"
}

FINISH

Make an async post request to transform-homo.heroku.com/ with the JSON object you formed earlier. You will be returned an array of transformed image.
